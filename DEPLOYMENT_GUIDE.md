## 🎯 **Quick Deployment Options**

### **Option 1: GitHub Pages (Free)**
1. Go to: https://github.com/gokulsenthilkumar3/Portfolio
2. Click **Settings** tab
3. Scroll to **Pages** section
4. Select:
   - Source: **Deploy from a branch**
   - Branch: **main**
   - Folder: **/ (root)**
5. Click **Save**
6. **Your site will be live at:** https://gokulsenthilkumar3.github.io/Portfolio

### **Option 2: Vercel (Recommended - Best Performance)**
1. Go to: https://vercel.com
2. Click **"New Project"**
3. Import: `gokulsenthilkumar3/Portfolio`
4. Click **"Deploy"**
5. **Your site will be live at:** https://portfolio-xyz.vercel.app

### **Option 3: Netlify (Alternative)**
1. Go to: https://netlify.com
2. Drag the `out` folder to Netlify
3. **Your site will be live instantly**

## 🔄 **Update Your Live Site:**

1. **Edit** `src/config/portfolio.config.ts`
2. **Run:** `npm run build`
3. **Commit:** `git add . && git commit -m "Update portfolio"`
4. **Push:** `git push origin main`
